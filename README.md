# Examination of Lightweight Linux GUI Distros for Small-Scale Private Cloud Deployment

## Project Overview

This repository contains the practical work and supporting documentation for the project entitled:

"Lightweight Linux GUI Distros for Small-Scale Private Cloud Deployment"

Three lightweight Linux distributions were evaluated:

- Lubuntu 26.04 LTS
- Xubuntu 26.04 LTS
- Linux Lite 7.8

The project investigated the suitability of these distributions for hosting lightweight private cloud services using Docker and nginx.

## Virtual Machine Configuration

All testing was performed in VMware using identical configurations:

- RAM: 4 GB
- CPU: 2 cores
- Storage: 20 GB
- Network: NAT
- Virtualization platform: VMware

## Practical Work

Docker was installed on each distribution and an nginx container was deployed to simulate a lightweight private cloud service.

Performance measurements included:

- Idle RAM usage
- Boot time
- Memory usage under Docker/nginx workload
- CPU utilisation
- Responsiveness and stability observations
