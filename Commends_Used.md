# Other Commands Used

## System Update

```bash
sudo apt update
sudo apt upgrade 
```

## Install htop

```bash
sudo apt install htop 
```

## Install Fastfetch

```bash
sudo apt install fastfetch 
```

## Install neofetch 

```bash
sudo apt install neofetch
```

## Measure Boot Time

```bash
systemd-analyze
```

## Measure Memory Usage

```bash
free -m
```

## Disk Usage

```bash
df -h
```

## Install Docker

```bash
sudo apt install docker.io -y
```

## Verify Docker

```bash
docker --version
```

## Run nginx Container

```bash
sudo docker start test-nginx
```

## View Running Containers

```bash
sudo docker ps
```

## Resource Monitoring

```bash
htop
```
